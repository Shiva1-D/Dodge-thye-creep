# Dodge the Creeps

## Description
Dodge the Creeps is an action-packed game where players must navigate through a maze while avoiding enemies.

## Changes Made
- Added shooting mechanic when the player scores 10 points.
- Improved enemy AI for better challenge.
- Enhanced graphics and animations for player and enemies.

## How to Play
- Use arrow keys to move the player.
- Press the "Fire" key to shoot bullets at enemies.

## How to Run
1. Open the project in Godot Engine.
2. Click "Play" to start the game.
